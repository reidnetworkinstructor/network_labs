5) Screenshot & Image Tips

• Use a clean terminal or UI theme for screenshots.
• Crop tightly to the relevant output or diagram.
• Include a short caption describing what’s shown.
• Save images in a dedicated /img or /docs folder.
Markdown example:
![Network Map](docs/network_map.png)
_A diagram generated from my network mapping lab showing VLAN segmentation._

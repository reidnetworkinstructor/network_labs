Hello, Python! Your First Script

Heads up: Try the lab first. When you’re ready, click to reveal answers.

Expected Output

Name: <Your Name>
Favorite Hoodie Skill: <Your Skill>
Date: 2025-08-10

Reference Code (hello.py)

print("Name: Jordan Kim")
print("Favorite Hoodie Skill: Cloud Security")
print("Date: 2025-08-10")

Why This Works

  • print() sends text to standard output; each call prints a new line.
  • Running python hello.py executes the file top‑to‑bottom as a script.
  • REPL is great for quick experiments; scripts are for repeatable results and versioning.

Common Mistakes & Fixes

  • Command not found: Python not on PATH → reinstall with “Add Python to PATH”, or run py on Windows / python3 on macOS/Linux.
  • Wrong folder: Terminal not in the project directory → run cd into the folder containing hello.py.
  • Quotes errors: Smart quotes from copy/paste → replace with straight quotes ".

Rubric (Pass/Complete)

  • Verified Python version in terminal (screenshot).
  • hello.py exists and runs without errors.
  • Output shows name, Hoodie skill, and today’s date.

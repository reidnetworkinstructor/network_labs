Why This Works

  • Variables store values in memory so they can be reused and manipulated.
  • int allows math operations; str is for text; bool represents logical state; float handles decimals.
  • f-strings make it easy to combine text with variable values in output.

Common Mistakes & Fixes

  • TypeError with concatenation: Trying to add int to str → use f-strings or convert with str().
  • NameError: Variable name typo → check spelling exactly.
  • Wrong output order: Print statements out of sequence → match the example for grading.

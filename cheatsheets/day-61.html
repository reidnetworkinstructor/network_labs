Basic Linux Navigation & Help Commands

Objective: Use the Linux terminal to navigate the file system, list files (including hidden ones), and look up command documentation with man.

ðŸ›  Tools Needed

  • Linux system (VM, WSL, or cloud instance)
  • Terminal access

Step 1: Check Your Current Location

Display the full path to your current directory:

pwd

Step 2: List Files in the Current Directory

List basic file names:

ls

List with detailed info:

ls -l

List all files including hidden ones:

ls -la

Step 3: Navigate Between Directories

  • Move to the system log directory:
    • cd /var/log
  • Go back to your home directory:
    • cd ~
  • Go up one directory from the current location:
    • cd ..

Step 4: Use man to Read Documentation

Read the manual page for the ls command:

man ls

Press q to exit the manual.

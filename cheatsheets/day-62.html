Day 62 Cheat Sheet

File and Folder Management

Quick reference for creating, renaming, moving, copying, and deleting files and directories in Linux.

Directory Commands

Command What it does Examples
mkdir Create a new directory mkdir project
mkdir -p reports/2025/january (nested)
cd Change directory cd /etc
cd .. (up one level)
ls List directory contents ls -l (long format)
ls -la (include hidden files)
rmdir Remove empty directory rmdir old_folder
rm -r Remove directory and its contents rm -r archive

File Commands

Command What it does Examples
touch Create an empty file or update timestamp touch notes.txt
cp Copy file or directory cp file1.txt file2.txt
cp -r dir1 dir2 (copy folder)
mv Move or rename file/directory mv old.txt new.txt (rename)
mv file.txt folder/ (move)
rm Delete file rm file.txt
rm -i file.txt (ask before deleting)

Wildcard Patterns

PatternMeaningExample
*Matches any charactersls *.txt
?Matches a single characterls file?.txt

Mini Workflows

# Create a project folder and files
mkdir project
cd project
touch notes.txt tasks.txt

# Rename and copy
mv tasks.txt todo.txt
cp notes.txt notes_backup.txt

# Move backup to another folder
mkdir archive
mv notes_backup.txt archive/

# Delete a file and folder
rm todo.txt
rm -r archive

Tips

  • Use ls -lh for human-readable file sizes
  • Add -i to rm for interactive safety
  • Combine commands with ; to run sequentially
  • Use man <command> for detailed options

Keyboard Shortcuts

  • Tab — Auto-complete names
  • Up/Down — Command history
  • Ctrl + C — Cancel current command

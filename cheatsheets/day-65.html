Day 65 Cheat Sheet

Package Management & Installing Tools

Quick reference for detecting your package manager, updating indexes, searching, installing, verifying, and removing packages across major Linux distributions.

Common Package Managers by Distro

DistroManagerNotes
Ubuntu/DebianaptModern replacement for apt-get
Fedora/RHEL/CentOSdnf / yumdnf is default on new versions
Arch/ManjaropacmanVery fast, binary package manager
openSUSEzypperHandles repos & dependencies well
AlpineapkLightweight, common in containers

Update/Refresh Index

ManagerCommand
aptsudo apt update
dnfsudo dnf check-update
pacmansudo pacman -Sy
zyppersudo zypper refresh
apksudo apk update

Search for Packages

ManagerCommand
aptapt search name
dnfdnf search name
pacmanpacman -Ss name
zypperzypper search name
apkapk search name

Install Packages

ManagerCommand
aptsudo apt install -y name
dnfsudo dnf install -y name
pacmansudo pacman -S --noconfirm name
zyppersudo zypper install -y name
apksudo apk add name

Remove Packages

ManagerCommand
aptsudo apt remove -y name • purge configs: sudo apt purge -y name
dnfsudo dnf remove -y name
pacmansudo pacman -R name • remove with deps: sudo pacman -Rns name
zyppersudo zypper remove -y name
apksudo apk del name

Verify Installation

# Check if installed and path
which pkgname

# Version check
pkgname --version

Service Management (systemd-based)

# Start service
sudo systemctl start servicename

# Enable on boot
sudo systemctl enable servicename

# Status
sudo systemctl status servicename --no-pager

Mini Workflows

# 1) Update catalog and install a tool
sudo apt update && sudo apt install -y htop

# 2) Search, install, verify, then remove
apt search htop
sudo apt install -y htop
which htop
sudo apt remove -y htop

# 3) Install and start nginx
sudo apt install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx
curl -I http://127.0.0.1

Tips

  • Always refresh indexes before searching or installing.
  • Use non-interactive flags (-y, --noconfirm) for scripting.
  • Purge (apt) or -Rns (pacman) to remove leftover configs.
  • Consider Snap or Flatpak for newer versions not in distro repos.

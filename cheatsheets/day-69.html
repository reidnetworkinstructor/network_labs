Day 69 Cheat Sheet

Disk & File System Management

Quick reference for viewing, mounting, and managing disks and file systems in Linux.

View Storage Devices

CommandPurpose
lsblkList block devices
lsblk -fList devices with file system info
sudo fdisk -lDetailed disk/partition info

View Mounted File Systems

mount | column -tList mounted file systems
df -TShow disk space with file system type
findmntTree view of mounts

Check Disk Space

df -hHuman-readable disk space
du -sh /pathDirectory size summary
du -ah / | sort -rh | head -n 10Top 10 largest files/dirs

Mount & Unmount

sudo mkdir /mnt/pointCreate mount point
sudo mount /dev/sdX# /mnt/pointMount device
sudo umount /mnt/pointUnmount by path
sudo umount /dev/sdX#Unmount by device

File System Maintenance

sudo fsck /dev/sdX#Check & repair file system (unmounted)
sudo mkfs.ext4 /dev/sdX#Create ext4 file system (erases data!)

Mini Workflows

# View all disks with FS info
lsblk -f

# Check disk usage
df -h

# Find largest files in /var
du -ah /var | sort -rh | head -n 10

# Mount a USB drive
sudo mount /dev/sdb1 /mnt/usb

# Unmount a USB drive
sudo umount /mnt/usb

Tips

  • Always unmount before running fsck to avoid corruption.
  • du shows space used by files/directories; df shows free/used space per filesystem.
  • Use mount -o loop file.iso /mnt/iso to mount ISO images.
  • File system creation commands will erase all data on the target partition â€” double check before running.

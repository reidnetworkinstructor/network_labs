Step 1: Use the Azure Pricing Calculator

  • Visit the Azure Pricing Calculator
  • Add the following services:
    • 1x Azure Virtual Machine (Windows, B2s, East US)
    • 1x Azure Storage Account
    • 1x Azure SQL Database (General Purpose)
  • Review the monthly cost estimation
  • Take a screenshot or note the breakdown of costs
  • Bonus: Change regions and observe cost differences

Step 2: Review Azure Cost Management + Billing

  • Log in to the Azure Portal
  • Search for Cost Management + Billing
  • Explore:
    • Cost analysis dashboard
    • Budgets (set one if possible)
    • Forecast tab
  • Understand how this helps monitor real-world Azure spending

Step 3: Analyze SLAs

  • Go to the Azure SLA page
  • Pick 3 services and record their SLA percentages (e.g., Azure VM, SQL DB, Blob Storage)

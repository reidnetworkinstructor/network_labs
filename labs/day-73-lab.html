Step 1 ‚Äì String Basics

Strings are text values. You can use single or double quotes, escape characters, and triple quotes for multi-line.

greet = "Hello"
who   = 'Hoodie'
line  = "He said \"hi\"."
block = """Multi-line
string example."""
print(greet, who)
print(line)
print(block)

Step 2 ‚Äì Concatenation vs f-Strings

Concatenation works, but f-strings are cleaner and safer.

first = "Jordan"
last = "Kim"

# Concatenation
full1 = first + " " + last

# f-string
full2 = f"{first} {last}"
print(full1)
print(full2)

Step 3 ‚Äì Getting Input

input() always returns a str. Convert types when you need numbers.

name = input("Your name: ")
year = int(input("Current year (e.g., 2025): "))
print(f"Hi {name}! Year recorded: {year}")

Step 4 ‚Äì Useful String Methods

Common tools you‚Äôll use constantly:

raw = "   Cloud SECURITY  "
clean = raw.strip().title()      # "Cloud Security"
lowered = clean.lower()          # "cloud security"
uppered = clean.upper()          # "CLOUD SECURITY"
parts = clean.split(" ")         # ["Cloud","Security"]
joined = "-".join(parts)         # "Cloud-Security"
replaced = clean.replace("Cloud", "Azure")
print(clean, lowered, uppered, parts, joined, replaced, sep=" | ")

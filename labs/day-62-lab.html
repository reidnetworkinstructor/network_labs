File and Folder Management

Objective: Learn to create, rename, copy, move, and delete files and directories using Linux terminal commands.

ðŸ›  Tools Needed

  • Linux system (VM, WSL, or cloud instance)
  • Terminal access

Step 1: Create a New Directory

Make a folder named project in your home directory:

cd ~
mkdir project

Step 2: Create Files

Inside the project folder, create two text files:

cd project
touch notes.txt todo.txt

Step 3: Rename a File

Rename todo.txt to tasks.txt:

mv todo.txt tasks.txt

Step 4: Copy a File

Make a copy of notes.txt called notes_backup.txt:

cp notes.txt notes_backup.txt

Step 5: Move a File to Another Directory

Create a folder named archive and move the backup file into it:

mkdir archive
mv notes_backup.txt archive/

Step 6: View Directory Contents

List contents in project and archive:

ls -l
ls -l archive

Step 7: Remove Files and Folders

Delete tasks.txt:

rm tasks.txt

Remove the archive folder and its contents:

rm -r archive

Collections: Lists & Tuples

Objective: Create and manipulate lists, understand indexing & slicing, practice common list methods, and learn when to use tuple for fixed data.

üõ† Tools Needed

  • Python 3
  • VS Code with Python extension

Step 1 ‚Äì Lists: your go-to, editable sequence

# Create a list
devices = ["router", "switch", "firewall"]
print(devices)              # ['router', 'switch', 'firewall']

# Indexing (0-based)
print(devices[0])           # router
print(devices[-1])          # firewall

# Slicing [start:stop:step] (stop is exclusive)
print(devices[0:2])         # ['router', 'switch']
print(devices[::-1])        # reversed copy

Step 2 ‚Äì List methods you‚Äôll use constantly

devices = ["router", "switch", "firewall"]

devices.append("ap")        # add to end
devices.insert(1, "ids")    # insert at index
print(devices)              # ['router','ids','switch','firewall','ap']

devices.remove("ids")       # remove first match
last = devices.pop()        # remove & return last ('ap')
print(last, devices)

devices.extend(["server","nas"])     # add many
devices.sort()               # alphabetical in-place
devices.reverse()            # reverse in-place
print(devices)

Step 3 ‚Äì Membership, counting, deduping

devices = ["router","switch","switch","ap","router","server"]
print("switch" in devices)       # True
print(devices.count("router"))   # 2

# Deduplicate while keeping order (simple approach)
unique = []
for d in devices:
    if d not in unique:
        unique.append(d)
print(unique)  # ['router','switch','ap','server']

Step 4 ‚Äì Tuple: like a list, but immutable (read-only)

Use a tuple when the collection should not change (e.g., coordinates, constant settings, fixed record).

# Tuple creation
coords = (29.7604, -95.3698)   # (lat, lon)
dns = ("8.8.8.8", "1.1.1.1")

# Index & unpack
print(coords[0])
lat, lon = coords
print(lat, lon)

# Immutability demo
# dns[0] = "9.9.9.9"   # ‚ùå TypeError: 'tuple' object does not support item assignment

Step 5 ‚Äì Useful patterns: enumerate & unpacking

devices = ["router","switch","ap"]
for i, name in enumerate(devices, start=1):
    print(f"{i}. {name}")

# Tuple unpacking in loops
inventory = [("router", 2), ("switch", 5), ("ap", 3)]
total = 0
for item, qty in inventory:
    total += qty
print("Total devices:", total)

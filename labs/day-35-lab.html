Day 35 Lab

Encrypt It Yourself: Mastering Hashing and Encryption with CyberChef

Objective: Use CyberChef to hash, encrypt, and decrypt messages — seeing how core cryptographic operations work in real time.

🛠 Tools Needed:

  • CyberChef (browser-based)
  • A simple text string (e.g., “HoodieSec2025!”)

Step 1: Create a Hash (SHA-256)

  • Go to CyberChef and enter: HoodieSec2025!
  • From the left menu, drag “To Hex” and then “SHA-256” to the recipe bar
  • Observe the output — a 256-bit hash value
  • Copy the result and paste it in your notes
  • Bonus: Try MD5 and SHA-1 — how do the outputs compare?

Step 2: Symmetric Encryption (AES)

  • Clear the recipe and use input: MySecurityLabIsAwesome
  • Drag “Encrypt” → AES → Set key: launchpad123
  • Choose “CBC” mode and enter IV: 1234567890123456
  • Copy your encrypted output
  • Now decrypt it by reversing the process
  • Reflection: What happens if the key or IV changes?

Step 3: Base64 Encoding

  • Drag “To Base64” to the recipe bar
  • See how the encrypted output changes format
  • This is a common method to store or transmit encrypted data

Step 4: Simulate Digital Signature (Hash + Public Key Idea)

  • Input a message: “Top Secret Mission: Hoodie Ops Alpha”
  • Hash it with SHA-256
  • Explain to a friend: Why is this unique? What if the message changes by 1 character?
  • Bonus Thought: This is the idea behind digital signatures — hash + encryption with a private key

Lab Summary: You just hashed data, encrypted it with AES, and experimented with encoding. These are the core building blocks of secure communication. CyberChef gives you a window into the math that protects your passwords, emails, and secrets every day.
